Differences between interpreted and compiled code

  • ON ERROR GOTO and ON BREAK GOTO will not compile correctly and must not be used on compiled programs.
  • PRINT statements will not compile correctly sometimes. Avoid to use functions and variables in print statements, which are not used anywhere else.
