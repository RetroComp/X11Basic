Das Programm kann mit folgenden Parametern aufgerufen werden:  

xbasic    --- Basic-Programm ausfuehren  [input.bas]

-l                  --- Programm nur laden 
-e        --- Basic Kommando ausfuehren 
--eval    --- Num. Ausdruck auswerten  

Beispiel:

xbasic testme.bas
xbasic -e 'alert 1,"Hallo !",1," OK ",b'
xbasic --eval 1+3-4*3
